# 🚀 SkyForce – 3D Space Shooter (Godot)

A 3D space shooter game developed using the Godot Engine, featuring enemy waves, shooting mechanics, particle effects, and interactive UI systems.

---

## 📌 Overview

SkyForce is an action-based space shooter game where the player controls a spaceship, eliminates incoming enemies, and survives increasingly difficult waves.

The project demonstrates core game development concepts such as **player controls, enemy AI, collision handling, UI systems, and shader effects** using the Godot Engine.

To play the game click on: https://skyforce1.netlify.app/

---

## 🎮 Features

- 🚀 Player-controlled spaceship  
- 🔫 Shooting mechanics with bullets  
- 👾 Enemy spawning system  
- 💥 Particle effects (afterburner, explosions)  
- ⏸️ Pause menu system  
- 🎵 Background music and audio integration  
- 🎨 Custom shaders for visual effects  
- 📋 Game menu UI  

---

## 🧠 Core Systems

- **Player System** → Movement, shooting, interaction  
- **Enemy System** → Spawning and behavior logic  
- **Game Manager (Global.gd)** → Game state handling  
- **UI System** → Menu and pause functionality  
- **Audio System** → Background music and effects  

---

## 🛠️ Tech Stack

- **Godot Engine**
- **GDScript**
- **Godot Shaders (gdshader)**
- **3D Assets (Space Shooter Assets Pack)**

## ⚙️ How to Run

1. Open **Godot Engine**  
2. Click **Import Project**  
3. Select the project folder  
4. Run the main scene
